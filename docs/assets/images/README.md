# Dualign Studio 截图资源索引

> 本文档是 `docs/assets/images/` 目录下所有截图的**权威描述**。
> 每张截图都有唯一的编号文件名，便于无多模态能力的 AI 或文档作者通过文字描述理解图片内容。

---

## 命名方案

文件名格式：`{序号:02d}-{英文描述}.png`

| 字段         | 说明                                                    |
| ------------ | ------------------------------------------------------- |
| **序号**     | 两位数字，反映截图在用户工作流中的展示顺序（01→11）     |
| **英文描述** | kebab-case，简洁描述界面状态或功能（如 `welcome-page`） |

> 重命名脚本：`rename_screenshots.py`（执行一次即可，有幂等保护）

---

## 通用规范

- **格式**：PNG
- **窗口尺寸**：统一 **1440×800**（16:9，此尺寸下各面板比例最佳）
- **系统**：Windows 11 风格
- **数据**：使用内置 Demo 数据（`Ctrl+D`）
- **截图范围**：仅 Dualign 主窗口内容区（不含外阴影、任务栏）
- **截图前确认**：Ollama 正常运行（欢迎页和底部栏指示灯均为绿色），API Key 已配置
